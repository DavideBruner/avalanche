Breakpoints

  • $breakpoint-xl
  • $breakpoint-s-max
  • $breakpoint-m-max
  • $breakpoint-l-max
  • $breakpoint-xl-max

Examples

Target min viewport width (default)

.box {
  width: 200px;

  @media (min-width: $breakpoint-s) {
    width: 300px;
  }
}

Target max viewport width (try to avoid)

.box {
  width: 200px;

  @media (max-width: $breakpoint-m-max) {
    width: 150px;
  }
}

Target a specific viewport width (try to avoid)

.box {
  width: 200px;

  @media (min-width: $breakpoint-s) and (max-width: $breakpoint-m-max) {
    width: 300px;
  }
}
